## Installation

Clone this repo and install all dependencies

```
 git clone https://github.com/ychengcloud/react-vite-admin.git

 cd react-vite-admin

 yarn install
```

## Development

``` 
yarn dev 
```

## Build

```
yarn build
```

## Publish

```
yarn dist
```
